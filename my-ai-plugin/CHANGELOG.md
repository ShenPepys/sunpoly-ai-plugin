# Changelog

## [0.1.0] - 2026-04-05

### 核心功能
- 多模型支持：DeepSeek / 豆包 / OpenAI / Claude 等 OpenAI 兼容 API
- 三种工作模式：Code（读写文件）、Ask（只读对话）、Plan（先规划后执行）
- 流式 SSE 对话，实时显示 AI 回复
- 工具调用系统：AI 可读写文件、列出目录，支持多轮自动调用
- Thinking 动画：AI 思考过程可视化
- 5 种代码命令：解释 / 修复 / 优化 / 续写 / 生成单测

### 聊天增强
- @ Mentions：输入 `@` 引用工作区文件作为上下文
- Slash 命令：输入 `/` 触发快捷命令
- 对话历史持久化（重启后不丢失）
- 输入历史回溯（↑↓ 箭头）
- 导出对话为 Markdown 文件
- Token 用量估算实时显示
- 错误重试按钮
- 上下文窗口管理（自动截断旧消息防止 Token 超限）

### UI 体验
- 代码块：语法高亮 + 复制/插入按钮 + 行号显示
- Markdown 表格渲染
- 停止生成（按钮 + Escape 快捷键，保留已接收内容）
- 复制整条 AI 回复（hover 显示）
- 消息时间戳
- 智能自动滚动
- 增强欢迎页（快捷键网格 + 功能提示）
- 输入框轮换提示文本
- 流式 Markdown 渲染优化（节流 + 未闭合代码块处理）

### 快捷键
- `Alt+Q` 聚焦聊天输入框
- `Alt+M` 切换工作模式
- `Alt+N` 新建对话
- `Alt+E` 解释代码
- `Alt+F` 修复代码
- `Escape` 停止生成

### 智能特性
- 状态栏模型指示器（切换自动同步）
- 工作区项目类型自动检测（增强 AI 上下文）
- API 双重超时（连接 30s + 空闲 60s）
- 429/401/403 友好错误提示
- 启动时 API 配置校验
- Webview 状态持久化（输入框 + 滚动位置）

## [0.2.0] - 2026-04-05

### 新增功能
- **Git 状态感知**：自动检测当前分支和未提交变更文件，注入 system prompt 增强 AI 上下文
- **网络代理支持**：HTTP/HTTPS 代理（CONNECT 隧道），兼容 VS Code 内置 `http.proxy` 配置
- **输入框字符计数**：实时显示输入字符数，超过 5000 字符时变色警告
- **拖拽文件到聊天**：从资源管理器拖拽文件到输入区域，自动添加为上下文，拖拽时高亮提示
- **聊天内搜索**：`Ctrl+F` 打开搜索栏，支持实时高亮匹配、上下导航（Enter/Shift+Enter）、Escape 关闭
