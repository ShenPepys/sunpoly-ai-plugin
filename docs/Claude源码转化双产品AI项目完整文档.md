## 三、产品一：AI 网页悬浮客服插件（高盈利款）

### 1. 产品核心信息

- **形态**：一行 JS 嵌入，右下角悬浮🤖图标，点击弹出聊天弹窗
- **适用场景**：国内企业 / 电商官网、外贸独立站、跨境商城
- **盈利模式**：国内版 99 元 / 月、299 元 / 年；外贸版 199 元 / 月、599 元 / 年；贴牌 / 私有化 5000-20000 元 / 套

### 2. 核心痛点与解决方案

| 核心痛点 | 解决方案 |
|---|---|
| 无自有模型无法 AI 对话 | 国内对接 DeepSeek / 豆包 API，外贸对接 OpenAI/Claude API |
| 用户跨页面咨询商品 | 商家后台自主配置独立知识库，AI 基于知识库回复，放弃爬虫 |
| 多客户数据串扰 | 客户唯一 ID（cid）+ 域名白名单，后端严格校验，知识库独立存储 |
| 国内外双场景适配 | 中 / 英双语知识库，自动切换对应模型与话术 |
| 商家无法实时接收咨询 | 全渠道消息转发，同步推送至国内外主流社交工具 |

### 3. 知识库存放与管控

- **存储位置**：服务商后端数据库，每个客户 ID 绑定独立知识库
- **配置方式**：商家后台自主编辑，支持中 / 英双语分开填写
- **调用逻辑**：插件加载校验通过后，仅调用当前客户专属知识库，无数据互通

### 4. 全渠道消息转发实现

#### （1）国内渠道

- **钉钉**：群机器人 Webhook，实时推送，稳定性最强
- **微信**：企业微信机器人 Webhook、公众号模板消息
- **旺旺**：无开放接口，间接转发至钉钉 / 企业微信，人工接管回复
- **短信 / 邮箱**：后端直接调用接口推送

#### （2）国外渠道

- **WhatsApp**：Meta 官方 WhatsApp Cloud API，支持双向互通
- **Facebook Messenger**：Meta 官方 API，同步至 FB 主页私信
- **X (Twitter)**：官方 API，私信实时同步推送
- **Instagram**：Meta 官方 API，同步私信消息
- **Telegram**：Telegram Bot 接口，实时提醒
- **Email**：自动邮件推送

#### （3）消息转发流程

```
用户网页发消息 → 后端接收 → AI 自动回复 → 转发至商家预设渠道 → 商家可人工接管
```

### 5. 商用权限管控

- **唯一标识**：每个客户分配专属 `cid`，JS 引入 `<script src="https://域名/robot.js?cid=xxx">`
- **校验逻辑**：后端校验 `cid` 有效性、绑定域名、授权状态、到期时间
- **隔离规则**：不同客户知识库、消息渠道、授权完全独立，杜绝盗用与串扰
- **停用机制**：到期未续费，后台修改状态，插件立即失效
